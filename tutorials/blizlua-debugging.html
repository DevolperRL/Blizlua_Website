Overview

This tutorial will guide you through the basics of creating a graphical user interface using ImGui with Blizlua. You’ll learn how to create your first ImGui window, manage its layout, and display simple UI elements.

By the end of this tutorial, you’ll have a solid understanding of how ImGui works, how to render windows in real time, and how to use Blizlua to build interactive and responsive user interfaces.
