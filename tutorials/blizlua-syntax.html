Blizlua Basics

Overview

BlizLua syntax is easy to learn and also understand. There is just a main.lua script, and from that script, you are going to load all other scripts for your add-on.

In BlizLua, you always start functions with the prefix bliz. or bliz.gl. From this prefix, you can easily access all the plugin's functions and see what parameters they require.
