Getting Started with Your First BlizLua Plugin

📋 Step 1: Download the Plugin

Download the plugin from the official BlizLua website: Click here

After downloading, place the plugin in the appropriate folder. Here are two examples:

General Plugin: X-Plane 12/Resources/plugins/
Aircraft-Specific Plugin: X-Plane 12/Aircraft/MyPlanes/Dc8/plugins/

📁 Step 2: Understand the Folder Structure

Your plugin folder should look like this:

📦 MyFirstPlugin/ ├── 📂 64/ ├── 📂 data/ │ ├── 📂 api/ │ ├── 📂 library/ │ │ └── 📄 yourLibrary.dll → Place all your .dll / .so / ... files here │ ├── 📂 scripts/ │ │ ├── 📂 module/ │ │ │ └── 📄 yourscript.lua → Your module scripts go here │ │ └── 📄 main.lua → Your main script │ └── 📄 config.cfg → Plugin configuration file └── 📂 license/

Note: If you place scripts in the wrong folder, the plugin might not detect them or they may not run correctly.

⚙️ Step 3: Configure config.cfg

Open the config.cfg file using any text editor (e.g., Notepad++).

Right now, the only required setting is the project name. Feel free to change it to whatever you like. Just remember to save the file afterward!

PROJECT_NAME=Test1

💻 Step 4: Edit main.lua

Time to test if everything works! Let’s start by adding a simple log message.

Note: BlizLua uses Log() instead of print().

function onFrame() -- Called every frame. See the manual: Documentation
    Log("Hello World")
end

🎮 Step 5: Test in X-Plane

Now that everything is set up, launch X-Plane and start a flight.

If everything went well, your plugin should print "Hello World" in the plugin log. Congrats — you’ve just created your first BlizLua plugin!

🪟 Creating Your First Custom Window

📋 Step 1: Add code to main.lua

Navigate to your main.lua file located inside BlizLua > data > scripts. To create a basic window, add the following code:

local win = bliz.CreateXPWindow({
    title = "Hello",            -- The title displayed on the window
    position = {300, 300, 500, 500}, -- {x, y, width, height}
    visible = true,             -- Determines if the window is shown
    background = true           -- Draws a background behind the window
})

In this example, a window titled "Hello" will appear at position x: 300, y: 300 with a size of 200x200 (from 300 to 500).

You can change the values of position to place and resize the window wherever you want.

🧪 Step 2: Run and test

Save the file, then launch X-Plane and start your flight. If everything is set up correctly, the custom window will appear automatically.

This window doesn’t do anything yet — but don't worry, we’ll add interaction in the next tutorial!

🪟 Drawing Inside Your Custom Window

In this part, you'll learn how to draw elements inside your custom window using BlizLua.

In BlizLua, drawing inside a window is only possible if you've created a window instance. Here's how to do it:

        win:Draw(function(id)
            -- See the manual for more functions: 
            -- https://devolperrl.github.io/Blizlua_Website/doc
            bliz.gl.DrawRectangle(id, 0, 0, 25, 25, true, 1, {1, 1, 1, 1}) 
            bliz.gl.DrawCircle(id, 250, 250, 50, 3, 5, false, {1, 1, 1, 1})
        end)

In this example, we draw a white rectangle and a circle inside the window. The id is the window's unique drawing ID.

You can explore more drawing functions (like lines, text, etc.) in the BlizLua documentation.

🧪 Step 2: Run and Test

Save your main.lua file, then open X-Plane and start your flight. If the window appears and you see the drawn shapes, everything is working perfectly!
