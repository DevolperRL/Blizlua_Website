BlizLua Documentation

Overview

BlizLua is a Lua scripting environment for X-Plane that provides:

  • Real-time script reloading
  • Access to X-Plane Datarefs and Commands
  • Custom drawing capabilities
  • File I/O operations
  • Network communication

Installation

  1. Download the latest package
  2. Extract to X-Plane/Resources/plugins/
  3. Create a scripts folder in the BlizLua directory
  4. Place your Lua scripts (.lua) in the scripts folder
  5. Start/Restart X-Plane

CreateXPWindow(title, position and size, visible, background)

Create a X-Plane window

Parameters:

  • title (string): Name of the window
  • position and size (table): Put here in a table the position and size of the window
  • visible (boolean): Set to true if you want the window to be visible
  • background (boolean): Set true if you want a background
win = bliz.CreateXPWindow({
                        title = "Hello",
                        position = {300, 300, 500, 500},
                        visible = true,
                        background = true,
                    })

Examples

Altitude Display

local altitude = xlua.find_dataref("sim/cockpit2/gauges/indicators/altitude_ft_pilot")

function draw()
    xlua.draw_text(50, 50, "ALT: " .. math.floor(altitude.value), 1, 1, 1, 1)
end

xlua.register_update(draw)

Script Reloading

Use xlua.reload_scripts() or create a keyboard shortcut:

xlua.register_command("sim/operations/reload_scripts", xlua.reload_scripts)
